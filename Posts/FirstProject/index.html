Project Overview

Easy Textil is a robust desktop application built to streamline the operations of a textile business. [cite: 2] Developed in Java [cite: 1], it features a clean and modern user interface thanks to the Swing GUI toolkit and the FlatLaf look and feel. [cite: 13]

The system provides a complete solution for managing clients, employees, products, and sales records. [cite: 2] All data is stored and managed in a MySQL database [cite: 14], and the application includes powerful reporting features powered by JasperReports to generate critical business insights.

Key Features

User Authentication

Secure login system authenticates users via CPF and password against a MySQL database. It supports different user roles with varying access levels.

Entity Management

Full CRUD (Create, Read, Update, Delete) functionalities for clients, employees, products, and sales records. [cite: 5, 6, 7, 8]

Database Administration

A dedicated menu for authorized users to create and delete the database [cite: 9] and its corresponding tables directly from the UI. [cite: 10]

Reporting System

Utilizes JasperReports to generate detailed reports for clients, employees, products, and sales. [cite: 11]

CEP Address Search

Includes a feature to automatically populate address fields by searching for a Brazilian postal code (CEP). [cite: 12]

Modern UI

Built with Java Swing and enhanced with the FlatLaf look and feel for a clean, modern, and responsive user interface. [cite: 13]
